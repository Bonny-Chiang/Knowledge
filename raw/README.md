# raw

`raw/` is the source-of-truth archive.

Store original materials here:

- PDFs
- webpages and webpage snapshots
- GitHub repositories or exported code archives
- images
- videos and subtitles
- datasets
- audio
- books
- miscellaneous incoming files

## Rules

- Do not manually edit original source files.
- Keep stable folders so AI agents can retrieve materials predictably.
- Add sidecar metadata files when needed, for example `paper-name.source.md`.
- Prefer preserving provenance: URL, author, date accessed, license, and capture method.

## Suggested Subfolders

- `inbox/`: newly captured items awaiting classification
- `papers/`: papers and academic PDFs
- `web/`: web articles, saved pages, snapshots
- `github/`: cloned or exported repositories
- `images/`: image sources
- `video/`: video files or video links
- `subtitles/`: transcript and subtitle files
- `datasets/`: datasets and data dictionaries
- `audio/`: audio files or audio transcripts
- `books/`: books and long-form documents
