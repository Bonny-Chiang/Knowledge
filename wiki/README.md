# wiki

`wiki/` is the Obsidian Vault and the durable knowledge layer.

Open this folder in Obsidian:

```text
Knowledge/wiki
```

## What Belongs Here

- Concept pages
- Method pages
- Person pages
- Project knowledge summaries
- Source summaries
- Maps of content
- Stable indexes

## What Does Not Belong Here

- raw PDFs or video archives
- project scratch work
- temporary notes
- experiment logs
- generated drafts
- code workspaces

Those belong in `raw/` or `workspace/`.

## Core Files

- [[index]]: top-level navigation
- [[log]]: chronological update log
- `_maps/`: maps of content
- `concepts/`: concept-centered notes
- `methods/`: method-centered notes
- `people/`: person-centered notes
- `projects/`: durable project summaries
- `sources/`: source summaries, not raw sources
- `tags/`: tag definition pages
