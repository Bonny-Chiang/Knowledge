# Promotion Workflow

Use this workflow when project work becomes durable knowledge.

1. Review the project folder in `workspace/active/`.
2. Identify stable concepts, methods, decisions, and reusable outputs.
3. Search `wiki/` for existing related notes.
4. Update existing notes when possible.
5. Create new notes only when the concept is genuinely new.
6. Add links back to project summaries or source summaries.
7. Add an entry to [[../log]].
