# 国发院政策建议稿写作规范

## Purpose

本方法页用于指导国发院内参/政策建议类稿件写作，尤其适合将调研、案例、数据和政策问题整理成可报送的决策参考文本。

## Selection Rules

- 选题要重大，具有现实政策意义。
- 注重时效，回应当前热点问题。
- 一事一议，避免一篇稿件承载过多议题。
- 可以围绕相关部门特定选题约稿展开。

## Content Rules

- 遵循科研诚信和学术规范。
- 语言要凝练。
- 数据要详实。
- 对策要务实。
- 每个观点都应有数据或案例支撑。
- 避免泛泛而谈和信息陈旧。
- 关键表述、数据、案例必须准确无误。
- 数据和案例应以脚注注明出处，包括网络链接、参考文献或具体资料。
- 如果无法获取或未使用最新数据，应以脚注说明原因。

## Length

- 常规政策建议：全文约 3000-4000 字，正文前附约 500 字摘要。
- 深度研究类报告：全文约 6000-8000 字，正文前附约 500 字摘要。

## Recommended Structure

```text
标题
作者信息
摘要
帽段/背景
一、基本情况
二、存在的问题
三、对策建议
第一作者联系方式
```

另一种样刊结构是：

```text
摘要
背景/调研情况
一、主要问题
二、深层次原因分析
三、对策建议
```

## Style Pattern

正文常用“一是、二是、三是”的并列论述方式，将事实、问题和建议压缩成清晰的政策表达。

## Submission and Review

- 投稿邮箱：`nads07@ruc.edu.cn`
- 约稿和报送联系：国发院成果转化中心工作人员。
- 审核流程：国发院内参编辑部审核 -> 学校政策研究室审核 -> 分管智库副校长审核 -> 学校党委书记审核。
- 工作日期间，录用情况通常每 2 周以邮件形式通知。
- 上级单位采用反馈时间一般为 2-3 个月，假期顺延。

## Reusable Checklist

- [ ] 题目是否重大、及时、一事一议？
- [ ] 摘要是否提炼出有观点性的关键判断？
- [ ] 帽段是否言简意赅交代背景和问题？
- [ ] 每个核心观点是否有数据或案例支撑？
- [ ] 关键数据、案例和表述是否有出处？
- [ ] 问题诊断是否避免泛泛而谈？
- [ ] 对策是否具体、务实、可执行？
- [ ] 字数和格式是否符合要求？
- [ ] 是否附第一作者联系方式？
