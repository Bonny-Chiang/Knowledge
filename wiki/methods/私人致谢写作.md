# 私人致谢写作

## Purpose

私人致谢不是正式论文致谢的替代品，而是一种阶段性自我整理：把一个阶段里重要的人、关系、支持和未竟之情保存下来。

## Method

1. 用一个具体场景开场，例如天气、地点、时间或身体动作。
2. 按关系场域组织人物，而不是按时间机械罗列。
3. 对每个场域写出共同经历和独特记忆。
4. 对关键人物保留一句有代表性的语言、动作或细节。
5. 允许不穷尽，承认“还有人未能写完”。
6. 最后回到家人和自我，完成阶段闭环。

## Pattern From Source

[[../sources/致谢二]] 的组织方式可以概括为：

```text
场景触发 -> 阶段回望 -> 人物群像 -> 关键支持 -> 旧友回声 -> 家人与自我
```

## Practical Use

如果未来要写毕业、离职、项目结束、城市告别或人生阶段复盘，可以沿用这个结构。
