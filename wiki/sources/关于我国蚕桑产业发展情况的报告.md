# 关于我国蚕桑产业发展情况的报告

## Source

- Raw file: `raw/papers/20240429关于我国蚕桑产业发展情况的报告.docx`
- Document type: 产业情况报告
- Institution: 乡村产业发展司（农产品加工指导司）
- Date in document: 2024-04-29

## Summary

这份报告系统梳理了我国蚕桑产业的发展现状、主要问题、既有支持政策和下一步工作方向。它将蚕桑产业界定为兼具传统优势、联农带农、生态价值和多元加工潜力的特色产业，同时指出机械化水平低、优质供给不足、综合利用不足、主体偏弱和国际竞争加剧等瓶颈。

## Key Findings

- 我国仍是全球蚕桑第一大国，2023 年桑园面积约 1137 万亩、蚕茧产量 76 万吨，分别约占世界的 75% 和 73%。
- 产业重心已明显西移，广西、四川、云南合计占全国桑园面积约 58%、蚕茧产量约 80%，蚕桑已成为部分地区的重要富民产业。
- 产业链正从“种桑养蚕缫丝”向桑叶、桑果、桑枝、蚕蛹、蚕沙等资源综合利用延伸，覆盖食品、饲料、生物医药、日化等多个领域。
- 主要短板集中在机械化智能化不足、缫丝产能结构性过剩、综合加工技术研发薄弱、经营主体分散和出口竞争压力上。
- 政策思路强调以全产业链方式推进，通过产业集群、产业强镇、技术攻关和龙头企业培育带动蚕桑产业高质量发展。
