# 关于补齐上海建设科技创新人才高地短板的建议

## Source

- 原始资料：`raw/papers/关于补齐上海建设科技创新型人才高地短板的建议-20221122.docx`
- 主题：上海科技创新中心、人才高地、青年科学家、高技能人才、国际人才政策

## Core Argument

这份材料认为，上海建设全球影响力科技创新中心的关键短板在于科技创新人才集聚能力不足，并将其表现为两类问题：上海在国家区域创新体系中的相对地位下降，以及海内外科技创新人才引留力度不足。

## Main Findings

- 上海发明专利授权数量和高精尖创新指数虽有增长，但与北京、深圳及部分长三角城市相比优势减弱。
- 上海对海外青年科学家、高端外籍人才的吸引力不足。
- 上海高学历人口、研发人员规模和占比与北京、深圳存在差距。
- 高端人才金融化可能挤压企业家型、科技创业型人才供给。

## Policy Suggestions

- 围绕集成电路、生物医药、人工智能等重点产业制定人才“金名单”。
- 依托张江科学城、临港新片区打造重大原始创新和国际化人才高地。
- 对青年科学家提供更大力度的资金、项目和跨界创新支持。
- 加强高技能人才培养，服务数字中国、网络强国和重点产业需求。
- 完善落户、住房、公共服务、跨区域人才认定等支持政策。
