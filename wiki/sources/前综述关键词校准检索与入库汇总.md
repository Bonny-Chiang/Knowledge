# 前综述关键词校准检索与入库汇总

## Source

- Raw file: `raw/github/research-review-01-keyword-calibration-search.md`
- Source type: AI workflow / skill document
- Scope: 中文社科项目前综述的关键词校准、CNKI/Google Scholar 检索、人工审核、Zotero 入库与 Obsidian 汇总。

## Key Claims

- 文献检索前必须先校准研究问题、关键词、学科语境和方法取向。
- 候选清单生成后必须暂停，等待人工审核；未获确认前不得进入 Zotero。
- “成功入库”的标准不是只有元数据，而是条目进入目标 collection 且 PDF 已附加。
- CNKI 与 Google Scholar 检索应优先使用专门工具链，普通网页搜索只能作为 fallback。
- Obsidian 阶段产出应是结构化汇总表，而不是逐篇精读笔记。
