# 合肥包河区低空政务巡查研究一页样品

## Source

- Raw file: `raw/papers/合肥包河区低空政务巡查研究一页样品.docx`
- Document type: 一页成果样品
- Topic: 政务无人机巡查需求识别、起降点适宜性和变量重要性。

## Summary

这份一页样品把包河区低空政务巡查研究压缩为可展示的成果页，核心输出包括巡查需求指数图、起降点适宜性图和随机森林变量重要性。它适合作为政策建议或项目汇报中的“成果样张”。

## Key Findings

- 潜在巡查需求较高区域主要集中在夜间活动强、人口较密集、POI 与道路设施集中的城市化片区。
- 需求指数范围为 0.2206-0.6208，北部和中西部部分网格较高。
- 相关分析中，夜间灯光 Pearson 0.803，人口密度 0.774。
- 随机森林测试集 R2 为 0.813，夜间灯光、人口密度和 POI 密度贡献靠前。
- 起降点适宜性前五位网格为 BH0253、BH0324、BH0286、BH0192、BH0335。
- 第一版适宜性网格只能作为候选片区，不能直接等同于正式建设点位。
