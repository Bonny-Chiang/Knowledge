# 国发院稿件样刊及说明

## Source

- Raw file: `raw/books/国发院稿件样刊及说明（新）.docx`
- Document type: 稿件说明 / 样刊模板 / 政策建议写作规范
- Institution context: 中国人民大学国家发展与战略研究院相关稿件报送

## Summary

这份文档说明了国发院政策建议/内参类稿件的选题、内容、篇幅、格式、投稿流程、审核流程和样刊结构。它不仅是格式模板，也隐含了一套政策建议写作方法：选题要重大及时，一事一议；观点要有数据或案例支撑；对策要务实；关键事实、数据和案例要可追溯。

## Key Requirements

- 选题重大、注重时效、一事一议。
- 服务当前热点问题、政策建议或决策依据。
- 遵循科研诚信与学术规范。
- 语言凝练、数据详实、对策务实。
- 每个观点应有数据或案例支撑。
- 关键表述、数据、案例必须准确，并以脚注注明出处。
- 常规篇幅约 3000-4000 字，正文前附约 500 字摘要。
- 深度研究类报告可扩展至 6000-8000 字。

## Suggested Structure

```text
标题
作者信息
摘要
帽段/背景
一、基本情况
二、存在的问题
三、对策建议
第一作者联系方式
```

样刊结构还包括“主要问题、深层次原因分析、对策建议”的问题导向写法。
