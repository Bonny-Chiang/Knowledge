# workspace

`workspace/` is the creation layer.

AI agents should do most day-to-day project work here.

## Structure

- `active/`: active projects
- `archive/`: completed or inactive projects
- `_template/`: project template

## Project Folder Contents

A project may contain:

- TODOs
- design documents
- prompts
- experiment logs
- scripts
- code
- reports
- generated outputs

## Rule

Do not put durable wiki pages directly in project folders. When project knowledge stabilizes, promote it into `wiki/`.
